# ChMS (Church Management System)

A minimalist, offline-first church management system for Africa. Built for reliability, speed, and practical core features.

## MVP Scope
- Organization Management
- Member Management
- Attendance (QR code, search/autocomplete, offline, minimal reporting)

See [pm/project-overview.md](pm/project-overview.md) for full details, roadmap, architecture, and workflow.

## Getting Started
1. Clone the repo
2. Install dependencies: `npm install`
3. Set up your `.env` and database
4. Run: `npm run dev`
