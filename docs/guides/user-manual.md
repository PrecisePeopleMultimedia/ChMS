## Getting Started

### First Login

1. Navigate to your ChMS URL (e.g., `https://your-church.chms.app`)
2. Click **"Login"** in the top right
3. Enter your email and password provided by your administrator
4. Click **"Sign In"**

### Forgot Password

1. Click **"Forgot Password?"** on the login page
2. Enter your email address
3. Check your email for password reset instructions
4. Follow the link to reset your password

### Profile Setup

After first login:

1. Click your profile icon (top right)
2. Select **"Profile Settings"**
3. Complete your profile:
   - Upload profile photo
   - Verify email address
   - Add phone number
   - Set notification preferences
4. Click **"Save Changes"**

## Authentication & Account Management

### Login Methods

#### Standard Login
- Enter email and password
- Click **"Sign In"**

#### Google OAuth (if enabled)
1. Click **"Sign in with Google"**
2. Select your Google account
3. Authorise ChMS access
4. You'll be automatically logged in

### Managing Your Account

#### Update Profile
1. Go to **Profile → Settings**
2. Edit any field:
   - Name, email, phone
   - Profile photo
   - Address
3. Click **"Save Changes"**

#### Change Password
1. Go to **Profile → Security**
2. Enter current password
3. Enter new password (twice)
4. Click **"Change Password"**

#### Notification Preferences
1. Go to **Profile → Notifications**
2. Toggle notifications:
   - Email notifications
   - Push notifications (mobile)
   - SMS notifications
3. Customise notification types:
   - New member registrations
   - Attendance reminders
   - Event updates
   - System alerts

## Member Management

### Adding a New Member

#### Method 1: Quick Add
1. Click **"Add Member"** on dashboard or Members page
2. Fill in required fields:
   - First Name, Last Name
   - Email (optional)
   - Phone (optional)
   - Date of Birth (optional)
3. Click **"Save"**

#### Method 2: Full Registration
1. Go to **Members → Add New Member**
2. Complete all sections:
   - **Personal Information**: Name, DOB, gender, etc.
   - **Contact Information**: Email, phone, address
   - **Family Information**: Assign to existing family or create new
   - **Member Type**: Member, Visitor, Child
   - **Custom Attributes**: Additional fields (if configured)
3. Upload profile photo (optional)
4. Click **"Save Member"**

### Searching Members

#### Quick Search
- Use search bar at top of page
- Search by: Name, email, phone, member ID
- Results appear as you type

#### Advanced Search
1. Go to **Members → Search**
2. Use filters:
   - Member Type (Member, Visitor, Child)
   - Active Status
   - Join Date Range
   - Custom Attributes
   - Tags/Badges
3. Click **"Search"**
4. Export results (CSV/Excel) if needed

### Editing Member Information

1. Find member (use search or browse list)
2. Click on member name or **"View Details"**
3. Click **"Edit"** button
4. Update information in any section:
   - Personal details
   - Contact information
   - Family assignment
   - Custom attributes
   - Badges/tags
5. Click **"Save Changes"**

### Member Profile Features

#### Overview Tab
- Basic information
- Contact details
- Profile photo
- Member status and type

#### Relationships Tab
- Family relationships
- Household memberships
- Relationship mapping
- Family tree view

#### Custom Attributes Tab
- Additional data fields
- Custom information (baptism date, etc.)
- External system IDs

#### Badges Tab
- Visual indicators
- Member tags
- Status badges

#### Notes Tab
- Member notes and history
- Pinned notes (always visible)
- Alert notes (high priority)
- Private notes (staff only)

#### Attendance Tab
- Attendance history
- Service attendance records
- Attendance patterns

### Member Actions

#### Deactivate Member
1. Open member profile
2. Click **"Actions" → "Deactivate"**
3. Confirm deactivation
4. Member marked as inactive (not deleted)

#### Reactivate Member
1. Search for inactive member
2. Open member profile
3. Click **"Actions" → "Reactivate"**
4. Member becomes active again

#### Merge Duplicate Members
1. Open first member profile
2. Click **"Actions" → "Merge"**
3. Search for duplicate member
4. Select fields to keep from each
5. Confirm merge

#### Export Member Data
1. Go to **Members → Export**
2. Select export options:
   - Member fields to include
   - Filter criteria
   - File format (CSV, Excel)
3. Click **"Export"**
4. Download file when ready

## Family Relationships

### Creating Families

#### Automatic Family Creation
- When adding a member, assign to existing family or create new
- Family name auto-generated from primary member

#### Manual Family Creation
1. Go to **Families → Add Family**
2. Enter family name
3. Add family members (select existing members or add new)
4. Set primary contact
5. Save family

### Managing Family Relationships

#### Adding Relationships
1. Open member profile
2. Go to **Relationships** tab
3. Click **"Add Relationship"**
4. Search for related member
5. Select relationship type:
   - Spouse
   - Parent
   - Child
   - Sibling
   - Guardian/Ward
   - Other
6. Add relationship details:
   - Start date
   - Notes
   - Custody information (if applicable)
7. Click **"Save Relationship"**

#### Editing Relationships
1. Open member profile → Relationships tab
2. Find relationship to edit
3. Click **"Edit"** icon
4. Update relationship details
5. Save changes

#### Viewing Family Tree
1. Open any family member profile
2. Go to **Relationships** tab
3. Click **"Family Tree"** view
4. Visual tree shows all relationships

### Household Management

#### Creating Households
1. Go to **Households → Add Household**
2. Enter household information:
   - Household name
   - Address
   - Phone
   - Household type (Primary, Secondary, Temporary)
3. Add household members
4. Set head of household
5. Save household

#### Managing Household Members
1. Open household profile
2. Click **"Members"** tab
3. Add/remove members:
   - Click **"Add Member"** to add existing member
   - Set member role (Head, Spouse, Child, etc.)
   - Set residency status (Current, Former)
   - Set residency dates
4. Save changes

## Attendance Tracking

### Recording Attendance

#### Method 1: QR Code Check-in (Recommended)
1. Go to **Attendance → Check-in**
2. Member scans QR code with mobile app
3. System automatically records attendance
4. Confirmation displayed

#### Method 2: Manual Entry
1. Go to **Attendance → Record Attendance**
2. Select service/event
3. Search for member
4. Click to mark attendance
5. Repeat for all attendees
6. Click **"Save Attendance"**

#### Method 3: Bulk Import
1. Go to **Attendance → Import**
2. Download template (CSV/Excel)
3. Fill in attendance data:
   - Member ID/Email
   - Service Date
   - Service Type
4. Upload file
5. Review imported records
6. Confirm import

### Service Management

#### Creating Services
1. Go to **Services → Add Service**
2. Enter service details:
   - Service name
   - Service type (Sunday Morning, Evening, Midweek, etc.)
   - Scheduled date and time
   - Location
   - Capacity (optional)
3. Configure options:
   - Allow family check-in
   - Require location assignment
   - Enable child security
4. Save service

#### Managing Services
- **Edit Service**: Update details, times, location
- **Duplicate Service**: Copy service for recurring events
- **Cancel Service**: Mark service as cancelled
- **View Attendance**: See who attended

### Viewing Attendance Records

#### Member Attendance History
1. Open member profile
2. Go to **Attendance** tab
3. View attendance timeline:
   - Date of service
   - Service type
   - Location
   - Attendance status

#### Service Attendance Report
1. Go to **Attendance → Reports**
2. Select service date range
3. Choose service types to include
4. View report:
   - Total attendance
   - Attendance by service type
   - Member attendance patterns
   - Trends over time
5. Export report if needed

## Mobile App Usage

### Mobile Features

- **QR Code Check-in**: Scan QR codes for attendance
- **Member Lookup**: Quick member search
- **Attendance Recording**: Manual attendance entry
- **Offline Mode**: Works without internet connection
- **Push Notifications**: Real-time updates

### Installing Mobile App

1. Download ChMS app from App Store or Google Play
2. Open app
3. Log in with your credentials
4. Allow required permissions (camera for QR scanning)

### Using QR Code Check-in

1. Open mobile app
2. Go to **Check-in** section
3. Point camera at QR code
4. System recognises member
5. Confirm attendance
6. Check-in complete

### Offline Mode

The app works offline:
- Previously loaded data accessible
- Attendance can be recorded
- Changes sync when connection restored

## Troubleshooting for Users

### Common Issues

#### Can't Log In
- **Check credentials**: Verify email and password
- **Forgot password**: Use password reset link
- **Contact admin**: If account locked or issues persist

#### Member Not Appearing in Search
- **Check filters**: Verify active status filters
- **Spelling**: Check spelling of name
- **Contact admin**: If member should be visible

#### Attendance Not Recording
- **Check service date**: Verify correct service selected
- **Internet connection**: Ensure online for sync
- **Contact admin**: Report persistent issues

#### QR Code Not Scanning
- **Camera permissions**: Ensure camera access granted
- **Lighting**: Improve lighting conditions
- **Distance**: Hold phone closer to QR code
- **Clean screen**: Remove smudges from camera

### Getting Help

1. **Documentation**: Check this manual
2. **Help Center**: Access in-app help section
3. **Contact Admin**: Email your system administrator
4. **Support**: Use support portal (if available)
